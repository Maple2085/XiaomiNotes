## ✨ 功能特性

- 🚀 **极速批量并发：** 采用异步并发下载技术，智能限速防封控，支持断点续传，千条笔记几分钟即可搞定。
- 🎨 **现代化归档：** 导出的 HTML 极为精美，自带**暗黑模式**、**全文实时搜索**、**字数统计**、一键复制，完美适配手机/平板/电脑端。
- 🔒 **绝对隐私安全：** 纯前端脚本执行，所有数据处理均在本地浏览器完成，无需上传至任何第三方服务器。
- ⚡ **智能格式保留：** 自动识别并保留图文混排、换行格式，支持加粗文字流式显示。

## 🛠️ 四步完美导出

无需安装任何繁琐软件，只需一个现代浏览器：

1. **登录云服务：** 电脑浏览器访问 [小米云服务官网 (i.mi.com/note/h5)](https://i.mi.com/note/h5) 并登录。
2. **打开控制台：** 按下 `F12` 键（或右键页面选择“检查”），切换到 **Console (控制台)** 标签页。
3. **粘贴并执行：** 复制本项目中的 `xiaomi-notes-exporter.js` 完整代码，粘贴进控制台，敲击 `Enter` 回车。
4. **下载归档：** 脚本将流式爬取数据，完成后会自动触发浏览器下载一个精美的本地归档 HTML 文件。

## ⚠️ 安全提示

* 本脚本仅建议在小米云服务官网控制台中执行。**请勿在其他陌生网站粘贴执行不明代码**，谨防账号信息泄露。
* 导出后的 HTML 文件为纯文本归档（保留原笔记排版结构）。
* 建议在私人电脑上操作，勿在公共设备登录你的云服务账号。
